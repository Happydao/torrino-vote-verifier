🔐 Torrino DAO Vote Verifier

Verify DAO votes independently using cryptographic wallet signatures.

📂 Where to get the data

You can find all voting data inside the public CSV files stored here:

👉 Torrino DAO Voting Data (GitHub)

Open a proposal CSV and copy:

- wallet
- signed_message
- signature

✍️ Input data

🧠 What this verifies

This tool checks that:

✔ The signature was produced by the wallet
✔ The message has not been altered
✔ The vote is cryptographically valid

⚠️ Important

This tool does NOT verify:

- NFT ownership
- Voting eligibility
- Duplicate votes
- Proposal validity or timing

🔐 Privacy

All verification happens locally in your browser.
No data is sent to any server.
